# StoryVote

![screen](https://github.com/prubianes/storyvote/assets/1655744/2b8a0855-2698-474e-8331-7a5d21424ca3)

StoryVote is an online voting system for scrum teams.
Developed using:
* Next.js
* Firebase
* Pico.CSS

Room Object in Firebase

```{
    rooms: {
        RoomId:{
            roomName
            votes,
            storyName,
            onlineUsers
        }
    }
}```
