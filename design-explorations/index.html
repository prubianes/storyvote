StoryVote / Concept Deck

Three visual directions for the next version.

Each concept is a full HTML/CSS-only exploration for the same product: room entry, voting interface, admin controls, and history. Open each option and choose the one worth turning into the real app.

Option 01

Modern Atelier

Editorial, warm, asymmetrical, and refined. Light-toned surfaces, copper accents, elegant typography, and layered glass forms.

Option 02

Brutalist Broadcast

Poster-grade hierarchy, exposed borders, industrial type, hard shadows, and loud contrast. Intentionally raw and anti-polished.

Option 03

Orbital Console

Deep-space command UI with scanlines, luminous data panes, technical labels, and system-status motion built from pure CSS.

Variant

Brutalist Broadcast Dark

A dark-mode counterpart to the brutalist route using red, cyan, blue, and mint only. No yellow in the palette.
